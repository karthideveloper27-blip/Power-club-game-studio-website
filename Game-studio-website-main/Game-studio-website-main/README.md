#  Game Over – Game Portfolio Website

Game Over is a fully responsive multi-page website project for a fictional game development company. It features beautifully designed pages including Home, About, Portfolio, News, and Contact.

The website highlights immersive storytelling, project showcases, global outreach, and community interaction. It’s designed using HTML, CSS, and Font Awesome for icons — offering a strong foundation for frontend development and creative web design.


##  Pages Included

- **Home** – Landing page with navigation and highlights.
- **About** – Information about the team, values, and creative process.
- **Portfolio** *(this page)* – Game projects, world-building examples, services, and reviews.
- **News** – Latest gaming updates and announcements.
- **Contact** – Contact form, address, social links.

##  Pages Overview

| Page | File | Description |
|------|------|-------------|
|  Home | `index.html` | Introduction to the studio with navigation links |
|  About | `about.html` | Company mission, team info, and design philosophy |
|  Portfolio | `portfolio.html` | Game projects, visuals, and testimonials |
|  News | `news.html` | Latest updates, blog posts, and articles |
|  Contact | `contact.html` | Contact form, social links, and company info |

##  Key Features

-  Clean, modern UI for gamers and developers
-  Responsive layout for all screen sizes
